THE PROCESS

To complete this WEBSITE, my team and I created wireframes to illustrate solutions for users’ needs.

For defining a set of visual elements used on the site we prepared a style guide.

Then we implement it with coding, and used GitHub to update data.

THE CHANLLAGE

a good experience

The biggest challenge is to design a clear visual hierarchy. A good hierarchical structure allows users to obtain information better and faster.

A pop-up effect on the navigation bar aims to confirm that the user's mouse in the right position.

We put the comment area and the writing review area in the same section according to the user’s scrolling down habits.

For a better browsing experience, A confirmation page will appear to tell the user they ordered successfully.

THE THINGS I LEARNED

In this project I learned how to use flexbox to design layout. I also learned that the design process should be based on real user experience, rather than cool style design. If there is a second chance to design this project. I will redesign the layout of the homepage because it looks too rigid.
